# Minerva
Microframework Typescript-only para construção de backends simples, projeto feito apenas para estudo do node e como funciona o pacote "http".

A Meta é suportar essas features 

1° Middlewares:
  O Mais simples, muito útil para modularizar o projeto, assim deixando com o dev se irá adicionar middlewares nas rotas, como json ou jwt por exemplo

2° Annotations:
  Quero um approach parecido com Spring, com annotations para simplificar a lógica de criação de rotas, por conta disso o js puro não irá funcionar.

3° Suporte ao Bun:
  Gostaria se possível suportar o bun nativamente, pois o Bun.Serve é mais rápido que o httpserver do node e tem uma melhor dev-experience que o node no geral.
